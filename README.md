# The Studio by Jonny

Personal barber website and installable mobile web app for Jonny, an Albuquerque-based USMC veteran.

## Included

- Responsive desktop and mobile layouts
- Booksy booking links and mobile call bar
- Service pricing, portfolio, policies, FAQ, and house-call information
- Custom The Studio by Jonny monogram and complete PWA icon set
- Branded offline screen and versioned service-worker cache
- Desktop pointer motion, scroll reveals, mobile menu transitions, and swipeable mobile portfolio
- Reduced-motion support

## Before publishing

Search `index.html` for `REPLACE` and add the final Booksy URL, Instagram handle, and phone number.

Upload every file and the `images` folder to the repository root. After replacing an older installed version, refresh the site once or remove and reinstall the Home Screen shortcut so the new icon and manifest are used.
