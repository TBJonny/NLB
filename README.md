# The Studio by Jonny

Personal barber website and installable mobile web app for Jonny, an Albuquerque-based USMC veteran.

## Included

- Responsive desktop and mobile layouts
- Booksy booking links and mobile call bar
- Service pricing, portfolio, policies, FAQ, and house-call information
- Custom The Studio by Jonny monogram and complete PWA icon set
- Branded offline screen and versioned service-worker cache
- Desktop pointer motion, scroll reveals, mobile menu transitions, and swipeable mobile portfolio
- Reduced-motion support
- Immediate mobile first screen with no blocking logo loader
- Built-in hamburger fallback that works even when an older external script is cached
- Versioned critical assets and fresh-first script/style updates
- Spring-based interactions and active-card settling on mobile

## Before publishing

Search `index.html` for `REPLACE` and add the final Booksy URL, Instagram handle, and phone number.

Upload every file and the `images` folder to the repository root. For this mobile repair, make sure `index.html`, `style.css`, `app.js`, and `sw.js` are all replaced together. After replacing an older installed version, refresh the site once or remove and reinstall the Home Screen shortcut so the new icon and manifest are used.
